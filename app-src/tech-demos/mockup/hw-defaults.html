Manage Holoseat Defaults

The four primary settings for Holoseat (shown below) have default values stored in the controller which persist when Holoseat is without power (e.g. it is unplugged or the computer is turned off). The values are not stored until you click the Save button. You may reset the values with the current defaults by clicking the Cancel button.
