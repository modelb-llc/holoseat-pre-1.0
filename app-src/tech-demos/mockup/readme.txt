This directory contains a static mockup of the Holoseat desktop app.  Use the
following command to start the Python webserver and run the mockup.

  python -m http.server

Inspired by wfdb-static-mockup by Alexander Putilin for "Mastering Flask" video
series from Packt Pub.  Source files can be found on Github:

https://github.com/eleweek/Mastering_Flask/tree/master/section3
