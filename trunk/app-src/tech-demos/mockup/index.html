Manage Holoseat Controls

Use theses settings to manage the current settings for the Holoseat controls. These settings will revert to the Hardware Defaults the next time Holoseat starts up. You can access Hardware Defaults the from the Settings menu.

These settings are updated live. Simply select a new value and Holoseat will update to reflect the new value.
